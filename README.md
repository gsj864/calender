# 📅 일정 캘린더 웹앱

자기소개 웹사이트에 연동되는 개인 일정 관리용 캘린더 웹앱입니다.

## ✨ 주요 기능

### 📆 캘린더 기능
- **월 단위 캘린더 뷰**: 직관적인 월별 캘린더 표시
- **날짜 선택**: 날짜 클릭 시 해당 날짜의 일정 확인
- **일정 표시**: 일정이 있는 날짜에 개수 배지 표시
- **월 이동**: 이전/다음 월로 쉽게 이동

### 📝 일정 관리 (관리자 모드)
- **일정 추가**: 제목, 설명, 날짜/시간, 타입, 공개 여부 설정
- **일정 수정**: 기존 일정 클릭하여 수정 가능
- **일정 삭제**: 삭제 확인 팝업 포함
- **일정 타입**: 면접 가능, 프로젝트, 개인 일정

### 🔐 권한 관리
- **방문자 모드**: 공개 일정만 조회 가능
- **관리자 모드**: 비밀번호(`gsj864`)로 접근, 모든 일정 관리 가능
- **공개/비공개 설정**: 일정별로 공개 여부 설정 가능

### ☁️ 데이터 저장
- **Firebase Firestore**: 클라우드 데이터베이스 연동
- **LocalStorage 백업**: 오프라인 지원 및 백업 저장

## 🚀 시작하기

### 필수 요구사항
- 웹 브라우저 (Chrome, Firefox, Safari, Edge 등)
- Firebase 프로젝트 및 Firestore 데이터베이스

### 설치 방법

1. **프로젝트 클론 또는 다운로드**
   ```bash
   # 프로젝트 폴더로 이동
   cd calendar
   ```

2. **Firebase 설정**
   - Firebase Console (https://console.firebase.google.com)에서 프로젝트 생성
   - Firestore 데이터베이스 생성
   - `config.js` 파일의 Firebase 설정 확인

3. **로컬 서버 실행** (선택사항)
   ```bash
   # Python 3가 설치되어 있는 경우
   python -m http.server 8000
   
   # 또는 Node.js가 설치되어 있는 경우
   npx http-server
   ```

4. **브라우저에서 열기**
   - `index.html` 파일을 브라우저에서 직접 열거나
   - 로컬 서버 주소로 접속 (예: `http://localhost:8000`)

### 📱 모바일에서 접속하기

모바일에서도 같은 일정을 보려면 **반드시 웹 주소(http/https)로 접속**해야 합니다.

1. **PC에서 로컬 서버 실행**
   ```bash
   python -m http.server 8000
   # 또는
   npx http-server -p 8000
   ```

2. **PC와 휴대폰을 같은 Wi‑Fi에 연결**

3. **PC의 IP 주소 확인**
   - Windows: `ipconfig` → IPv4 주소 (예: 192.168.0.5)
   - Mac: 시스템 설정 → 네트워크에서 IP 확인

4. **휴대폰 브라우저에서 접속**
   - 주소창에 `http://PC의IP:8000` 입력 (예: `http://192.168.0.5:8000`)

5. **Firebase 연동이 안 될 때**
   - Firebase Console → 프로젝트 설정 → **사용자 인증 정보** → **승인된 도메인**
   - 여기에 접속하는 주소 추가 (예: `192.168.0.5` 또는 `localhost`)

> ⚠️ `file://`로 열거나, PC와 휴대폰이 다른 네트워크면 모바일에서 접속·연동이 되지 않습니다. 항상 같은 Wi‑Fi + `http://PC_IP:포트`로 접속하세요.

## 🎨 사용 방법

### 방문자 모드
1. 페이지를 열면 기본적으로 방문자 모드로 시작
2. 캘린더에서 날짜를 클릭하여 해당 날짜의 공개 일정 확인
3. 월 이동 버튼(◀ ▶)으로 다른 월 확인

### 관리자 모드
1. 우측 상단의 "방문자 모드" 버튼 클릭
2. 비밀번호 입력 모달에서 비밀번호 `gsj864` 입력
3. 관리자 모드로 전환되면:
   - 일정 추가 버튼이 캘린더 아래에 표시됨
   - 모든 일정(공개/비공개) 조회 가능
   - 일정 클릭 시 수정/삭제 가능

### 일정 추가
1. 관리자 모드로 전환
2. "일정 추가" 버튼 클릭 또는 날짜 선택 후 추가
3. 일정 정보 입력:
   - 제목 (필수)
   - 설명 (선택)
   - 날짜 (필수)
   - 시작/종료 시간 (선택)
   - 일정 타입 선택
   - 공개/비공개 설정
4. "저장" 버튼 클릭

### 일정 수정/삭제
1. 관리자 모드에서 일정 클릭
2. 수정 모달에서 정보 변경 후 "저장"
3. 또는 "삭제" 버튼으로 일정 삭제

## 🐛 문제 해결

### Firebase 연결 오류
- Firebase Console에서 Firestore 데이터베이스가 생성되었는지 확인
- `config.js`의 Firebase 설정이 올바른지 확인
- 브라우저 콘솔에서 에러 메시지 확인

### 데이터가 표시되지 않음
- 브라우저 콘솔에서 에러 확인
- LocalStorage에 백업 데이터가 있는지 확인
- Firebase 보안 규칙 확인

### 일정이 저장되지 않음
- 관리자 모드로 전환되어 있는지 확인
- Firebase Firestore 권한 확인
- 네트워크 연결 상태 확인
